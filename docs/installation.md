# Installation

## Stable release

To install autoBind, run this command in your terminal:

```sh
uv add autoBind
```

Or if you prefer to use `pip`:

```sh
pip install autoBind
```

## From source

The source files for autoBind can be downloaded from the [Github repo](https://github.com/mane292/autobind).

You can either clone the public repository:

```sh
git clone git://github.com/mane292/autobind
```

Or download the [tarball](https://github.com/mane292/autobind/tarball/master):

```sh
curl -OJL https://github.com/mane292/autobind/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd autobind
uv pip install .
```
